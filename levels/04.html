Mais de uma classe pode ser aplicada ao mesmo elemento! No HTML, basta inserir um espaço entre cada classe e, no CSS, um ponto entre cada classe.

Por exemplo, o elemento <a class="card sucesso"> pode ser selecionado com o seletor .card.sucesso. Este seletor seleciona apenas os elementos que possuam as duas classes juntas.
