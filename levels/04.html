Mais de uma classe pode ser aplicada (de uma vez só!) ao mesmo elemento! No HTML, basta inserir um espaço entre cada classe e, no CSS, um ponto entre cada classe.

Por exemplo, o elemento <a class="origem fala"> pode ser selecionado com o seletor .origem.fala. Este seletor seleciona apenas os elementos que possuam as duas classes juntas.
